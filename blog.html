Question-2: What is the difference between innerHTML, innerText, and textContent?
  • innerHtml returns all text, including html tags and browser render correctly
  • innerText returns all text, ignores html tags.
  • textContents returns only text, ignores HTML tags. It deals with raw text and doesn't concern about styles
Question-3: What is event delegation in the DOM?
Event delegation is a technique, where event listener is apply on parent element. That event is also used to handle child element. As a result perfomance is good and no need to apply event listner for adding others elements
Question-4: What is event bubbling in the DOM?
After clicking element, using capturing phase it goes to down.When gets the target element, is called target phase. Using 'bubbles up' it goes to upward.
